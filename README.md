# Confession Agent

A Streamlit web application that acts as a "Legal Tamil Confession Generator" using the Google Gemini API.
The application converts conversational Tamil/Tanglish inputs into formal "Bulletproof" Legal Tamil Confessions (Opputhal Vaku Moolam).

## Usage
1. Clone the repository.
2. Install the requirements using `pip install -r requirements.txt`.
3. Provide your Gemini API key in the `.env` file or directly through the UI sidebar.
4. Run the application:
```bash
streamlit run app.py
```
